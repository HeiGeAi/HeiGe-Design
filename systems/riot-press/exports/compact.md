# 暴走印刷（riot-press · 凶悍工业）— Compact

> RiotPress is an underground xerox-zine identity: hazard yellow #ffe600 slapped over photocopier black on gray copy-stock paper. Ultra-condensed all-caps display type stacks like flyposted headlines, mono stamps date every block, and crooked torn-tape labels carry the attitude. Hard offset shadows, zero radius, static half-tone grain. One token set drives both the 界面 and the 16:9 演示. No gradients, no soft corners, all bite.

**Theme：** light

## 用前先读（5 条铁律）

1. 主色 `#ffe600` 只给关键 CTA 与强调，克制使用。
2. 复现完整 DESIGN.md 里 `## Signature Moment` 写死的签名时刻。
3. 中文字体栈末尾必带系统兜底，中文不用日文字体渲染。
4. 不碰紫蓝俗套渐变，不做居中卡片三件套。
5. 交付前对照 `## Do's and Don'ts` 自查，跑 `heige-design lint riot-press` 确认 0 error。

> 完整设定（组件规格 / 图像语言 / 演示系统 / Agent 配方卡）见同目录 `DESIGN.md`。
