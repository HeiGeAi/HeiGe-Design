# 熔炉（forge-anvil · 凶悍工业）— Compact

> An aggressively industrial, high-contrast dark system for devtools and infrastructure. Iron-black canvas, one acid-green electric voltage, concrete-gray structure. Archivo Black manifesto headlines over JetBrains Mono data, hard non-blurred offset shadows, and exposed machine-bed grid rulers. Built to boot cold and hit hard.

**Theme：** dark

## 用前先读（5 条铁律）

1. 主色 `#c8ff00` 只给关键 CTA 与强调，克制使用。
2. 复现完整 DESIGN.md 里 `## Signature Moment` 写死的签名时刻。
3. 中文字体栈末尾必带系统兜底，中文不用日文字体渲染。
4. 辉光只用 box-shadow / radial-gradient，不用 backdrop-filter，不动画化阴影。
5. 交付前对照 `## Do's and Don'ts` 自查，跑 `heige-design lint forge-anvil` 确认 0 error。

> 完整设定（组件规格 / 图像语言 / 演示系统 / Agent 配方卡）见同目录 `DESIGN.md`。
