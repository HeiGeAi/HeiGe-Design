# HeiGe-Design 气质图谱（Aesthetic Atlas · 50 套原创品牌设定集）

这是 HeiGe-Design 的选美层。50 套**原创虚构品牌**，全部为 HeiGe 原创，不逆向、不复刻任何真实品牌，规避商标与视觉识别侵权。

来源基因：HeiGe-UI 八大气质方向 + HeiGe-PPT 六大气质方向 + 12 套 UI 样例 + 6 套 deck 样例的真实配方，向外扩展成覆盖全球设计正典的 11 个家族，另加一整个西方库没有的**东方新中式家族**当差异化护城河。

每套给出：`slug` / 中文名 / 品牌人格 / 行业适配 / 调色板方向（含 hex 种子）/ 字体方向 / 签名手法 / 标签。这份是 Phase B 生成 50 份 DESIGN.md 的直接输入。

选美铁律（对齐 HeiGe 反 AI 清单）：
- 每套走一个**极端**方向，宁可有记忆点，不做"现代简约"这种安全废话。
- 主色 + 至多两个辅助色 + 中性色，节奏靠版式不靠堆色。
- 绝不碰紫蓝烂大街渐变（#667eea→#764ba2）及其全家桶。
- 每套必须有一个换个模板绝对做不出来的**签名时刻**。
- 中文字体栈必带系统兜底，中文绝不用日文字体渲染。

## 家族 01 · 凶悍工业 Brutalist（5 套）

硬核、高对比、不讨好、有攻击性。近黑 + 一个高饱和电光色，粗黑边框、硬阴影（offset 不模糊）、裸露网格。种子样例：forge-landing / streetwear-drop / brat-noisepark。

### forge-anvil · 熔炉
- 人格：开发者工具的宣言体，像一台冷启动的重型机床。
- 行业：devtools / 基础设施 / 开源项目 / 硬科技。
- 调色板：铁黑 #0c0d0e + 酸绿电光 #c8ff00 + 混凝土灰 #8a8d8f，纯白反差字。
- 字体：Archivo Black / Space Grotesk 超粗做标题，JetBrains Mono 做 kicker 和数据。
- 签名：超大字号宣言占满一屏 + 裸露网格线 + 硬 offset 阴影（`6px 6px 0 #000`）。
- 标签：brutalist / devtool / high-contrast / acid。

### rebar-concrete · 钢筋
- 人格：施工现场的秩序美学，警示与承重。
- 行业：工业 SaaS / 建筑科技 / 制造 / 供应链。
- 调色板：混凝土灰 #b8b5ad + 警示橙 #ff5b1f + 沥青黑 #17181a。
- 字体：Stencil 感无衬线（Anton 压缩体）+ 等宽标注。
- 签名：警示斜条纹分隔带 + 承重网格 + 巨型编号。
- 标签：industrial / caution / stencil / structural。

### riot-press · 暴走印刷
- 人格：地下丝网印刷厂，复印机噪点，态度先行。
- 行业：潮牌 / 独立媒体 / 音乐厂牌 / 亚文化社区。
- 调色板：漆黑 #000 + 警示黄 #ffe600 + 复印灰噪点。
- 字体：超大压缩体（Bebas / Druk 感）全大写，贴纸胶带混排。
- 签名：xerox 噪点质感 + 歪贴的胶带标签 + 一句占屏狠话。
- 标签：streetwear / xerox / poster / punk。

### ironclad-ledger · 铁账
- 人格：给硬核工程师用的金融工具，锈铁与齿轮。
- 行业：面向开发者的 fintech / 交易基础设施 / B2B 结算。

### brut-signal · 呐喊
- 人格：抗议海报的能量，木刻版画的粗粝。
- 行业：活动 / 公益倡导 / 有主张的品牌 / 宣言页。
